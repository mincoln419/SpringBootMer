뷰 / GET index 페이지

/sign-up GET 회원가입 페이지 　
/mypage GET 회원정보 수정 페이지

/bulletin GET 자유게시판 페이지 　
/calculator GET 형량계산기 페이지 　
API /api/account GET 계정 전체 조회 계정 관리 - 관리자, 사용자 로 나누어 권한관리
/api/account POST 계정 신규 생성
/api/account/{account-id} GET 계정 단일 조회
/api/account/{account-id} PUT 계정 단일 수정
/api/account/{account-id} DELETE 계정 단일 삭제
/api/notice GET 공지사항 조회 형량계산기 운영관련 공지사항
/api/notice POST 공지사항 신규 생성
/api/notice/{notice-id} GET 공지사항 단건 상세 조회
/api/notice/{notice-id} PUT 공지사항 단건 상세 수정
/api/notice/{notice-id} DELETE 공지사항 단건 삭제
/api/notice/{notice-id}/reply GET 공지사항 댓글 전체 조회
/api/notice/{notice-id}/reply POST 공지사항 댓글 신규 생성
/api/notice/{notice-id}/reply/{reply-id} GET 공지사항 댓글 단건 조회
/api/notice/{notice-id}/reply/{reply-id} PUT 공지사항 댓글 단건 수정
/api/notice/{notice-id}/reply/{reply-id} DELETE 공지사항 댓글 단건 삭제
/api/bulletin GET 게시판 조회 형량 계산기에 대한 개선사항, 실제사례 등에 대한 토론, 현재 법현실에 대한 비판적 논의를 할 수 있는 공간
댓글 이외에 답글 기능 - 게시판 사이에 tree layer를 두고 답글 구성 가능
/api/bulletin POST 게시판 신규 생성
/api/bulletin/{bulletin-id} GET 게시판 단건 상세 조회
/api/bulletin/{bulletin-id} PUT 게시판 단건 상세 수정
/api/bulletin/{bulletin-id} DELETE 게시판 단건 삭제
/api/bulletin/{bulletin-id}/reply GET 게시판 댓글 전체 조회
/api/bulletin/{bulletin-id}/reply POST 게시판 댓글 신규 생성
/api/bulletin/{bulletin-id}/reply/{reply-id} GET 게시판 댓글 단건 조회
/api/bulletin/{bulletin-id}/reply/{reply-id} PUT 게시판 댓글 단건 수정
/api/bulletin/{bulletin-id}/reply/{reply-id} DELETE 게시판 댓글 단건 삭제
/api/biz GET 비즈니스 이벤트 조회 사실관계 + 적용법령에 따라 나온 결과
/api/biz POST 비즈니스 이벤트 신규 생성
/api/biz/{biz-id} GET 비즈니스 이벤트 단건 상세 조회
/api/biz/{biz-id} PUT 비즈니스 이벤트 단건 상세 수정
/api/biz/{biz-id} DELETE 비즈니스 이벤트 단건 삭제
/api/biz/{biz-id}/sit GET 사실관계 조회 주체(가해자), 객체(피해자), 수단, 피해결과, 정상참작사유 등의 요건을 근거로 구성요건 성립, 위법성 조각, 책임유무, 형량감경 등을 판단 + 주요 판례들의 형량을 나타냄
/api/biz/{biz-id}/sit POST 사실관계 생성
/api/biz/{biz-id}/sit PUT 사실관계 수정
/api/biz/{biz-id}/sit DELETE 사실관계 삭제
/api/biz/{biz-id}/law GET 적용법령 전체 조회 일반법, 특별법 적용 가능여부 판단(또는 선택)
/api/biz/{biz-id}/law POST 적용법령 생성
/api/biz/{biz-id}/law/{law-id} GET 적용법령 단건 조회
/api/biz/{biz-id}/law/{law-id} PUT 적용법령 수정
/api/biz/{biz-id}/law/{law-id} DELETE 적용법령 삭제
